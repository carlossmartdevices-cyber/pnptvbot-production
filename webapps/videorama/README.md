# PNPtv Videorama (React)

Mobile-first interface for:
- YouTube links (video + playlist)
- Video uploads via existing backend `POST /api/videorama/upload`

## Dev

```bash
cd webapps/videorama
npm install
npm run dev
```

## Build (static)

```bash
cd webapps/videorama
npm install
npm run build
```

Copy `webapps/videorama/dist` to `public/videorama-app/` and the app will be available at:
- `/videorama-app/`
